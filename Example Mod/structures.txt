-----------------------------------------------------------------
The Mod structure in mods folder should always be 
-----------------------------------------------------------------

    mods\YourMod_Example\..      
_________________________________
|    - poster aka thumbnail     |
|    - icon                     |
|    - mod.info                 |
_________________________________
    ..\media\                   
_________________________________
|     - stuff...                |
_________________________________

-----------------------------------------------------------------
The Collection structure in workshop folder should always be 
-----------------------------------------------------------------

    Workshop\YourCollection_Example\..                   
________________________________________________________
|    - preview.png                                     |
|    - workshop.txt (Generates after first upload)     |
________________________________________________________
    ..\Contents\mods\                                 
________________________________________________________
|        - YourMod_Example1 (Folder)                   |
|        - YourMod_Example2 (Folder)                   |
|        ...                                           |
________________________________________________________

-----------------------------------------------------------------

Usually if your mod is not loading/appearing in game, 
you messed something with that or mod.info file.

That's why when you install for example 10 years later MOD, 
you have 6 version of it, because it is collection of mods.
